# Moon

A Little CPU Singlethreaded Raytracer written in Rust

## Current Capabilities

- Lambertian shading
- Blinn-Phong shading
- Hard Shadows
- Reflections
- Single Light Source
- Colored Light
- Export to .pbm file

## In The Future

- Multithreading
- GPU Acceleration
- Export to more file formats (.png, .jpeg)
- Complex Shapes
- Texturing
- Anti-Aliasing
- Soft Shadows
- Multiple Light Sources
- Light Refraction
- Scene file
- And More..
