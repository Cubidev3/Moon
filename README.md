# Be-a-Ray

A Little Raytracer written in Rust
Attention!: This project is under heavy development

# Current Capabilities

- Lambertian shading
- Blinn-Phong shading
- Hard Shadows
- Reflections
- Single Light Source
- Colored Light
- Export to .pbm file
